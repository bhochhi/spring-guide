Spring-security
----
Spring Security is a framework that focuses on providing both authentication and authorization to java application. Authentication is the process of establishing a principal who they claim to be( a "principle" generally means a user, device or some other systems which can perform an action in your application). "Authorization" refers to the process of deciding whether a principal is allowed to perform an action within your application.
